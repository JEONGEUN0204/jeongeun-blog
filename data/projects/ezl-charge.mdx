`MAU 30만 규모`의 교통카드 충전·조회 서비스이며, React Native 기반의 프로젝트입니다.
사용자들이 모바일 환경에서 편리하게 교통카드를 충전하고 이용 내역을 조회할 수 있도록 안정적이고 직관적인 서비스를 제공했습니다.

✔ 이즐워크에서 API 연속 호출 문제를 개선하여 불필요한 호출을 줄이고 `리소스를 절감함`과 동시에 `사용자 경험을 향상`시켰습니다.<br/>
✔ 충전·환불 과정에서 미완료된 거래를 사용자가 재시도할 수 있는 플로우를 구현하여 관련 `CS 인입을 감소`시켰습니다. <br/>
✔ Sentry를 활용해 주요 에러를 모니터링하고 해결함으로써 `앱의 크래시율을 효과적으로 낮췄습니다.`
